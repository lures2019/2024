#### 文章评定

![](./imgs/读者.png)

![](./imgs/精力.png)

写文章就是表达自己的观点，尽可能提供多的信息量，但是又不消耗读者过多能量的过程。



#### 论文书写

拿到文章后，会先看图和表(最直观的)，大部分审稿人是不会看附录的

一般实验部分占到`40%`左右，最少要`30%`，会让别人觉得你工作量比较大

![](./imgs/摘要.png)

不要完全叙述模型做了什么，要表现出模型的亮点

![](./imgs/asbtract.png)

![](./imgs/修改前的摘要.png)

开头的不必要，摘要中尽量不要写太长的话。
